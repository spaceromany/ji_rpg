# RPG 전율 (RPG Jeonyul)

Godot 4.5로 개발 중인 턴제 RPG 게임 프로젝트입니다. 옥토패스 트래블러 스타일의 전투 시스템과 실시간 필드 맵을 결합한 게임입니다.

## 주요 기능

### 필드 맵 (Field Map)
- 실시간 유닛 이동 및 조작
- 드래그 선택으로 다중 유닛 선택
- 우클릭으로 이동 명령 (다중 선택 시 진형 유지)
- 마우스 휠로 줌 인/아웃
- 지원 범위 시각화 (Support Range)
- 적과 조우 시 자동 전투 전환

### 전투 시스템 (Battle System)
- **턴 기반 전투**: 속도(SPD) 기반 턴 순서 결정
- **BP 시스템**: 부스트 포인트로 스킬 위력 강화 (R키로 부스트)
- **브레이크 시스템**: 적의 쉴드를 부수면 턴 순서 맨 뒤로 이동
- **스킬 시스템**: 공격, 회복, 버프/디버프 스킬

### UI 시스템
- 턴 순서 표시 (현재 사이클 + 다음 사이클 미리보기)
- 배틀러 상태 표시 (HP, SP, BP, 쉴드)
- 커맨드 UI (공격/스킬/방어)
- 타겟 선택 UI
- 데미지 팝업

## 프로젝트 구조

```
새-게임-프로젝트/
├── assets/              # 게임 리소스 (음악 등)
├── scenes/              # Godot 씬 파일
│   ├── field/          # 필드 맵 관련 씬
│   ├── game/           # 게임 매니저 씬
│   ├── ui/             # UI 씬
│   └── test_battle.tscn # 전투 테스트 씬
├── scripts/             # GDScript 파일
│   ├── audio/          # 오디오 관리
│   ├── battle/         # 전투 로직
│   ├── data/           # 데이터 구조
│   ├── field/          # 필드 맵 로직
│   ├── game/           # 게임 관리
│   └── ui/             # UI 로직
└── project.godot       # Godot 프로젝트 설정
```

## 조작법

### 필드 맵
| 입력 | 동작 |
|------|------|
| 좌클릭 | 유닛 선택 |
| 좌클릭 드래그 | 범위 선택 |
| 우클릭 | 선택된 유닛 이동 |
| 마우스 휠 | 줌 인/아웃 |

### 전투
| 입력 | 동작 |
|------|------|
| 방향키/WASD | 메뉴 이동 |
| Enter/Space | 선택 |
| ESC | 취소/뒤로 |
| R | 부스트 (BP 사용) |
